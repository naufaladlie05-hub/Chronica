CHRONICA adalah game LOCAL PARTY yang mengajak pemain untuk menjadi ahli dalam membuat kebohongan!!!
Dalam game ini di setiap rondenya akan ada pertanyaan dimana para pemain harus membuat jawaban palsu dari pertanyaan tersebut yang dapat mengecoh pemain lainnya!!! 
Hanya ada 1 jawaban yang benar sehingga para pemain harus saling menipu untuk mencapai skor tertinggi!!!
